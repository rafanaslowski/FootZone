# FootZone MarketPlace

O FutZone é uma plataforma de marketplace focada exclusivamente no universo dos tênis (sneakers). O objetivo é conectar compradores e vendedores, oferecendo uma experiência fluida, segura e estilosa para quem busca desde modelos casuais até as edições limitadas mais raras do mercado.
## Funcionalidades

### Para Compradores
Busca Avançada: Filtros por marca, tamanho, cor e faixa de preço.

Autenticação: Sistema de login seguro para gerenciar pedidos e favoritos.

Carrinho Dinâmico: Adicione e remova itens com atualização de valor em tempo real.

Checkout Integrado: Simulação de pagamento e cálculo de frete.

### Para Vendedores
Dashboard de Vendas: Gráficos de desempenho e histórico de transações.

Gestão de Inventário: Cadastro de produtos com upload de fotos e descrição técnica.
